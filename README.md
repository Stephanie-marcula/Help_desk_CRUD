# HelpDesk Pro

HelpDesk Pro é um sistema CRUD desenvolvido em PHP e MySQL para gerenciamento de chamados técnicos.

O projeto simula um módulo de controle de chamados dentro de um sistema de Help Desk, permitindo cadastrar, listar, editar e excluir solicitações, além de controlar prioridade e status dos atendimentos.

## Demonstração

Acesse a versão online:

https://helpdeskpro.freedev.app

> Sistema publicado apenas para fins de portfólio.  
> A versão demonstrativa possui limite de 50 chamados cadastrados.

## Funcionalidades

- Cadastro de chamados técnicos
- Listagem de chamados
- Edição de título, descrição, prioridade e status
- Exclusão de chamados
- Controle de status
- Controle de prioridade
- Proteção com token CSRF
- Modo demonstração com limite de registros

## Tecnologias utilizadas

- PHP
- MySQL
- PDO
- HTML5
- CSS3
- Bootstrap
